# Toronto Housing Market Analysis

This project analyzes housing affordability in Toronto using public data.

## Goals
- Explore rental and housing trends in Toronto
- Clean and combine public datasets
- Visualize key market patterns
- Build a simple regression model to study drivers of affordability

## Tools
- Python
- pandas
- matplotlib
- scikit-learn
- Jupyter Notebook

## Data Sources
- City of Toronto Open Data
- Toronto Housing Data Hub
- CMHC Rental Market Data
